# My-Engine-Projects

A collection of game projects, graphical simulations, and game prototypes built entirely from the ground up using a **custom, low-level C++ game engine**. 

This repository serves as a timeline of my engine architecture journey, shifting from initial rendering primitives to robust subsystem management, custom memory handling, and real-time HLSL shader effects.

## 🛠️ Tech Stack
* **Language:** C++ (99.3%)
* **Graphics API:** DirectX 11
* **Shading Language:** HLSL (0.7%)
* **Platform:** Windows PC (x64)

## 🏗️ Building and Setup

### Prerequisites
* **OS:** Windows 10/11 (64-bit)
* **IDE:** Visual Studio 2022 (with the **Desktop development with C++** workload installed)
* **SDK:** Windows 10/11 SDK (Includes DirectX 11 headers)

### Compilation
1. Clone the repository natively.
2. Open the main solution file (`.sln`) in Visual Studio.
3. Switch your build configuration to **Release / x64** for optimized execution.
4. Build the solution (`Ctrl + Shift + B`) and launch your desired startup project.
