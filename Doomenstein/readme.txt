Doomenstein

* Game was made for mouse and keyboard single player

Movement
- W / Left Stick Up - Move forward
- S / Left Stick Down - Move backward
- A / Left Stick Left - Strafe left
- D / Left Stick Right - Strafe right
- Mouse / Right Stick - Look around
- Shift / A Button - Sprint

Combat
- Left Mouse / Right Trigger - Shoot
- 1 / X Button - Weapon Slot 1
- 2 / Y Button - Weapon Slot 2 ( If filled )

Interaction
- Space / X Button - Interact with Weapons Chest
- Space / X Button - Use Pack-a-Punch machine

Game
- ESC / Back - Return to menu
- ~ - Toggle open command

Tips

- Keep moving - Demons spawn from within the walls
- Use the Weapons Chest to get random powerful weapons
- Pack-a-Punch upgrades make weapons significantly stronger
- Different weapons excel at different ranges
- Use AddGold=X command to add gold
